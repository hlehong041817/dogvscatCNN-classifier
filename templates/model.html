The Model

This classifer was based on the model: MobileNetV2. The train dataset contains 12,500 images of cats and 12,500 of dogs. The model archieved ~98% accuracy on train set and ~98% on validation set.

The final Dense layer uses Softmax activation function, it returns a probablity of the sample to be a cat. For that reason, this classifer is only useful to differentiate between dogs and cats.
